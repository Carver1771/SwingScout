# SwingScout

A marketplace to find and book golf instructors across Dallas–Fort Worth.

**Live at:** [swingscout.com](https://swingscout.com)

## Stack
- Single-page HTML/CSS/JS
- Leaflet.js for interactive maps
- Deployed on Vercel

## Updating
Edit `index.html` and push to main. Vercel auto-deploys.
